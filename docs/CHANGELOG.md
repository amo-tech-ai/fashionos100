
# Changelog

## [0.9.0] - 2025-03-03
### ✨ Added - Complete Sponsorship Suite
- **Sales & CRM:**
  - `SponsorDetailPage`: Comprehensive view with contracts and history.
  - `EventOpportunitiesPage`: Inventory management for event sponsorships.
  - `DashboardLeads`: Pipeline tracking for potential sponsors.
  - `DashboardPackages`: Management of sponsorship tiers.
- **Refactor:**
  - Split monolithic `SponsorshipPages.tsx` into atomic files for better maintainability.
  - Updated routing in `App.tsx` to support deep linking (e.g., `/sponsors/:id`).
