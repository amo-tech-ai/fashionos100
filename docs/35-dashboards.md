# Service Dashboards

## 1. Photography Dashboard
*   **Kanban:** Pre-prod -> Shooting -> Retouching -> Delivery.
*   **Gallery:** Proofing grid with "Favorite/Reject" actions.

## 2. Video Dashboard
*   **Timeline:** Production phases.
*   **Review:** Frame-by-frame comment tool for video drafts.

## 3. Web/Ecom Dashboard
*   **Site Health:** Uptime, Speed.
*   **Store Stats:** Sales, Conversion Rate, Top Products.
*   **CMS:** AI-assisted blog editor.

## 4. Social/Marketing Dashboard
*   **Planner:** Calendar view of posts.
*   **Analytics:** Engagement charts, ROI tracker.
*   **Asset Bank:** Approved visuals for posting.

## 5. Event Dashboard
*   **Command Center:** Countdown, Budget, Critical Alerts.
*   **Logistics:** Map view of venue, Guest list status.
*   **Run of Show:** Live timing controller.
