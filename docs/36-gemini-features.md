# Gemini 3 Pro Feature Map

| Capability | FashionOS Application | Real-World Example |
| :--- | :--- | :--- |
| **Text Generation** | **Briefs & Copy** | Drafting a press release for a show. Writing 50 product descriptions. |
| **Gemini Thinking** | **Planning Logic** | Resolving schedule conflicts between a model's fitting and the rehearsal. |
| **Function Calling** | **Actions** | `createEvent()`, `bookModel()`, `sendInvoice()`, `checkAvailability()`. |
| **Search Grounding** | **Research** | Finding trending colors for the season. Researching potential sponsors. |
| **Maps Grounding** | **Logistics** | "Find a venue near [Hotel] with a loading dock." |
| **Structured Outputs** | **Data Integrity** | Converting a messy client email into a structured `Task` list JSON. |
| **URL Context** | **Brand DNA** | scraping `client.com` to extract primary colors and tone of voice. |
| **File Search (RAG)** | **Knowledge** | Answering questions from a 50-page PDF Venue Contract. |
| **Code Execution** | **Calculators** | Calculating complex taxes and service fees for a $100k budget. |
